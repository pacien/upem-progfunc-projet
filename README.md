# UPEM / L3 Info / Functional programming / Project: URM

Unlimited Register Machine in OCaml.

## Usage

* Requirements: ocaml, GNU Make, OCaml Makefile
* Build: `make nc` produces a binary named `urm`
* Clean: `make clean`
